Web SQL admin

Web SQL是HTML5中新增的接口，本程序可以用来管理webSQL，适合SQL新手用来学习和测试SQL脚本。

数据完全保存在你的本地计算机中，无需网络连接也可正常使用。

太好了! 你的浏览器支持webSQL接口，现在开始使用吧。
不好了! 你的浏览器不支持webSQL接口，建议更换为 Chrome浏览器 。或者尝试在手机上打开本页面。一般的安卓浏览器均支持此接口。

在下方运行SQL

  • 显示表数据
  • 插入数据
  • 修改数据
  • 删除行
  • SELECT *
  • WHRER
  • ORDER BY
  • INSERT INTO
  • 删除表
